What is Fabric?

  • Official Docs: http://www.fabfile.org/
  • Python 2.5-2.7 library and command line tool that let's you write Python instead of shell scripts for SSH and sysadmin tasks

What is Ansible?

  • Official Docs: http://docs.ansible.com/
  • Ansible is an agentless, open source automation platform.

Playbooks

  • YAML (Yay!)
  • Define your deployment steps and configuration
  • Modular, contain variables
  • Mature modules you can lean on
